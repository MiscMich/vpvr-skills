## On-Page SEO Checklist

### Title Tag Optimization

| Element | Requirement | Example |
|---------|-------------|---------|
| Length | 55-60 characters | ✅ "Luxury Florida Keys Vacation Rentals | Marathon FL" |
| Keyword | Near front | ✅ "Marathon FL Vacation Rentals..." |
| Brand | At end | ✅ "...| Villa Paraiso" |
| Power Word | Include one | ✅ "Best", "Luxury", "Private", "Waterfront" |

**VPVR Title Formula**:
```
[Primary Keyword] + [Location] + [Differentiator] | Villa Paraiso
```

### Meta Description

| Element | Requirement |
|---------|-------------|
| Length | 150-160 characters |
| Keyword | Include naturally |
| CTA | Action-oriented |
| Unique Value | Why book with VPVR |

**VPVR Meta Description Formula**:
```
[Benefit] + [Primary Keyword] + [Location]. [Unique Value]. [CTA]
```

Example:
```
Book luxury waterfront vacation rentals in Marathon, Florida Keys. Private pools,
docks & ocean access. Save 15-20% booking direct. Explore our Keys properties →
```

### Header Hierarchy

| Tag | Purpose | SEO Value |
|-----|---------|-----------|
| H1 | Page title (one only) | Highest |
| H2 | Major sections | High |
| H3 | Subsections | Medium |
| H4+ | Details | Low |

**H1 Rules**:
- One H1 per page only
- Contains primary keyword
- Matches search intent
- 40-60 characters ideal

### URL Structure

| Good URL | Bad URL |
|----------|---------|
| `/vacation-rentals/marathon-fl/` | `/p?id=12345` |
| `/property/blue-pearl/` | `/property/12345-rental-house-florida-keys-marathon-pool/` |
| `/blog/fishing-marathon-florida-keys/` | `/blog/2024/01/15/fishing-article/` |

**VPVR URL Rules**:
- Lowercase only
- Hyphens between words
- 3-5 words maximum
- Include primary keyword
- No dates in URLs

## Answer Engine Optimization (AEO)

### The AEO Difference

| Traditional SEO | Answer Engine Optimization |
|-----------------|---------------------------|
| Rank on page 1 | Be the cited answer |
| Click to website | Answer shown directly |
| Keyword matching | Question answering |
| Link building | Entity optimization |

### Content Structure for AI Extraction

**First 100 Words Rule**:
Every page must contain a complete, standalone answer in the first 100 words.

```markdown
## What Are the Best Vacation Rentals in Marathon FL?

The best vacation rentals in Marathon, Florida Keys are waterfront properties
with private pools, docks, and ocean access. Marathon offers 4-bedroom luxury
homes like Blue Pearl (70ft dock, heated pool) and Emerald Oasis (kayaks,
canal-front). These properties combine Florida Keys charm with resort-level
amenities at 15-20% less than hotel alternatives when booked direct.
```

### Featured Snippet Formats

**Paragraph Snippet (40-60 words)**:
- Triggers: "What is", "Why does", "How does"
- Format: Direct, complete answer

**List Snippet (5-10 items)**:
- Triggers: "Best", "Top", "How to", "Steps to"
- Format: Numbered or bulleted list

**Table Snippet (3-6 rows)**:
- Triggers: "vs", "comparison", "prices"
- Format: Clean table with headers

### Voice Search Optimization

| Traditional Query | Voice Query |
|-------------------|-------------|
| "vacation rentals Marathon FL" | "Where can I stay in Marathon Florida?" |
| "Florida Keys fishing spots" | "What's the best place to fish near Marathon?" |
| "Keys vacation pool dock" | "Are there vacation rentals with pools and docks in the Keys?" |

**Voice Optimization Rules**:
1. Use conversational, natural language
2. Answer questions directly
3. Include question in H2 headers
4. Keep answers speakable (40-60 words)

### AI Citation Optimization (2026)

Get cited in Google AI Overviews, Perplexity, and Claude.ai:

| Strategy | Implementation | Example |
|----------|----------------|---------|
| **Direct Answers** | First sentence answers the query completely | "The best vacation rentals in Marathon FL are waterfront homes with pools and docks." |
| **Entity Clarity** | Name specific properties, locations, features | "Blue Pearl offers a 70-foot dock" not "our properties have docks" |
| **Structured Lists** | Use numbered/bulleted lists for steps | "3 steps to book: 1) Browse, 2) Check dates, 3) Book direct" |
| **Comparison Tables** | Create scannable comparison data | Hotel vs vacation rental table |
| **FAQ Schema** | Mark up Q&A pairs | Every blog should have 3-5 FAQs |

**Zero-Click Optimization**:

65%+ of searches now end without a click. Optimize for brand visibility even without clicks:

1. **Featured Snippet Format**: 40-60 word paragraph answers
2. **Knowledge Panel**: Complete Google Business Profile
3. **Local Pack**: Optimize for "vacation rentals near me"
4. **Image Pack**: Alt text with location + property type
5. **People Also Ask**: Target related questions in content

### Multi-Platform AEO

Optimize for all AI answer engines, not just Google:

| Platform | How to Optimize |
|----------|-----------------|
| **Google AI Overviews** | Structured data, FAQ schema, E-E-A-T signals |
| **Perplexity AI** | Clear factual statements, cited sources, updated dates |
| **Claude.ai** | Comprehensive content, logical structure |
