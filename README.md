## Need a New Skill?

Use the **Skill Creator** (`skill-from-masters`) to build skills based on proven expert methodologies.

1. Add the skill-from-masters skill to your Claude
2. Ask: "Help me create a skill for [your task]"
3. It will find relevant expert frameworks and create a high-quality skill
